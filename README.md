# Finetuning-GLiNER-Turkish-Span-Based-NER

GLiNER Turkish Medical NER
Bu proje, GLiNER (Generalist Model for Named Entity Recognition) mimarisini kullanarak Türkçe takviye edici gıda (vitamin & supplement) yorumlarından varlık ismi çıkarma (NER) işlemini gerçekleştirmektedir. Model, çok dilli urchade/gliner_multi-v2.1 modeli temel alınarak fine-tune edilmiştir.
